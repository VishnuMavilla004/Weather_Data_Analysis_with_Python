# 🌦️ Weather Data Analysis using Python

## 📌 Project Overview

This project performs exploratory data analysis (EDA) on weather data using Python in Google Colab. It focuses on identifying patterns, trends, and insights related to temperature, humidity, and weather conditions.

---

## ⚙️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

---

## 🔍 Key Analysis Performed

* Data Cleaning and Preprocessing
* Handling Missing Values
* Exploratory Data Analysis (EDA)
* Temperature and Humidity Analysis
* Weather Condition Trends
* Data Visualization

---

## 📊 Key Insights

* Identified temperature variations across time
* Analyzed humidity patterns
* Observed frequency of weather conditions

---

## 📁 Project Structure

* `notebook/` → Jupyter Notebook
* `data/` → Dataset (if applicable)

---

## 🚀 How to Run

1. Open the notebook in Google Colab or Jupyter
2. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
3. Run all cells
